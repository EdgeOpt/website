Objectives

The objective of the course is to train the professionals of your organisation to understand Artificial Intelligence (AI) and Machine Learning (ML). At the end of the training session, your professionals can be effectively resourced in various AI/ML projects in your organisation to transform your business.

Course Outline

We will introduce you to the world of Artificial Intelligence (AI) and Machine Learning (ML) in an extremely simple and practical way.

  • Introduction
    • Basics: Natural intelligence vs. artificial intelligence, surogate model, digital twin.
  • Supervised and Unsupervised learning
    • Classification: classification problem, decision tree, classification rules, classification of example elements, memory and its use; Linear and nonlinear regression: parametric model of regression, delta rule, linear and nonlinear models, approximation, neural networks (multilayer perceptron, network parameters).
    Regression: Time series modeling, digital twin, probabilistic modeling.
    Advanced: Vision: Image classification, Natural Language Processing: Open-source language models, Clustering.
  • Reinforcement learning Reinforcement learning tasks, Markov decision processes, stochastic strategies, dynamic programming, Q-learning, use of reinforcement learning.
    •
Since Python is the most famous language used in the area of AI/ML, we will give you an introduction on writing Python codes for the purpose of building machine learning models.

  • Getting Started: data types, tuples, strings, loops, lists.
  • Intermediate: built-in functions, indexes and slices, functions, function parameters and typing.
  • Advanced: Numpy, Pandas, mutability, error handling, iterators/iterables, package creation.
    •
This part will be highly interactive. We understand your business, and introduce right learning sources from where you can start learning. We will also introduce you to potential opportunities in your own sector (e.g., Aeronautics, Aerospace, Computer Science, Bio, Industrial, Solution Architect, etc.) Some of the other topics to be covered include:

  • Open-source data
  • Open-source models
  • Open-source source code repositories
    •
In this part, we will be discussing about the necessary computer setup needed to start building AI/ML models.

  • Editors such as Vscode, Pycharm, etc.
  • Shell scripting using Terminal, Git, Dvc, CI/CD
  • Gitlab, Containers such as Docker
  • Machine Learning IDEs, Clusters such as Kubernetes
  • Installation of Scikit-Learn, TensorFlow, PyTorch
    •
Finally, you will be using the skills which you gathered from the previous parts of the course in order to actually develop AI/ML models for practical purposes.

  • Introduction to Scikit-Learn, TensorFlow, PyTorch
  • Introduction to MxNet
  • Introduction to Tensorflow Agents
    •
You will learn about bringing the developed machine learning model to production so that it can be used by users/apps in real-life. We will use the following technolgies:

  • Git, DVC, Docker
  • Gitlab CI/CD, Dashboards, Rest API, AWS
    •
In this section, we will provide you with real-life AI projects where you get to work and gain knowledge on the end-to-end spectrum of building a machine learning product: data mining, data processing, model building, model evaluation, and building a simple app. We will do the project as it is usually done in the companies.

  • Jira (tasks/tickets creation) for AI project management
  • Sprint style Agile software development
  • Gitlab for source code hosting
  • Python for machine learning model developemnt
  • Docker for container development
  • CI/CD and RestAPI for app development.
